# Help for `man.q`


:fontawesome-brands-github:
[KxSystems/man](https://github.com/kxsystems/man)

## Examples
```txt
man "$"               / operator glyph
man "enum extend"     / operator name
man "read0"           / keyword
man ".z"              / namespace
man "-b"              / command-line option
man "\\b"             / system command
```


## Special pages
```txt
man ""                / reference card
man "cmdline"         / command-line options
man "errors"
man "datatypes"
man "debug"
man "interfaces"
man "internal"
man "iterators"
man "kb"              / Knowledge Base
man "syscmds"         / system commands
man "wp"              / White Papers
```

## Arguments to `man`
```txt
man "--list"
man "--help"
```
