# `til`



Syntax: `til x`, `til[x]` 

_First x natural numbers_ 

Where `x` is a non-negative integer, returns the first `x` integers. 
```q
q)til 0
`long$()
q)til 1b
,0
q)til 5
0 1 2 3 4
q)til 5f
'type
  [0]  til 5f
       ^
```
