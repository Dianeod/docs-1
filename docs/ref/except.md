## `except`



_Exclude items from a list_

Syntax: `x except y`, `except[x;y]`

Where 

-   `x` is a list 
-   `y` is a list or atom

returns all items of `x` that are not (items of) `y`.

```q
q)1 2 3 except 2
1 3
q)1 2 3 4 1 3 except 2 3
1 4 1
```
