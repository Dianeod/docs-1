# `sqrt`




_Square root_

Syntax: `sqrt[x]` (prefix, unary, atomic)

Returns the square root of numeric `x`. Where `x` is negative or null, the result is null.

```q
q)sqrt -1 0n 0 25 50
0n 0n 0 5 7.071068
```
